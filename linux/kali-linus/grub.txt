1. To disable dhcp protocol while installation : 
    - `netcfg/use_dhcp=false` on boot
2. If you want to use a local repository mirror instead of http.kali.org, you can pass its name on the kernel command line (at boot-time) 
    with a syntax like this: mirror/http/hostname=my.own.mirror.
3. `Debconf` is a configuration system for Debian packages. 
4. Kali installation for ARM/embedded devices like raspberripi. Refer 'scripts/kali-arm-master.zip'
