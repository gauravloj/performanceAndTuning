1. Footprinting using nmap:
    - nmap.org provids services to scan legally using nmap.
    - `nmap -v -A scanme.nmap.org` : It scans 'scanme.nmap.org' for any open ports and other network related info
    - `nmap -oG scanme.nmap.org` : Displays grepable output
    - `nmap -oG -p 22 scanme.nmap.org` : Scans on port 22
    - `nslookup scanme.nmap.org` : It queries internet domain name server
    - `nslookup 74.207.224.221` : It reverse lookup the IP for mapped DNS. reverse IP is stored as 221.2224.207.71.in.addr.arpa
    - Search 'major ip block' to get list of important IPs around the world
    - Use grep or awk to extract IP that are scannable i.e. IP with open ports
    - Eg. - `nmap -oG scanme.nmap.org -vv | grep Up awk -F ' ' '{print $2}'`
